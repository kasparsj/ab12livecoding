# Remix Performance Study

Advanced course notes and a runnable teaching skeleton inspired by
[`lil-data/vroomvroom`](https://github.com/lil-data/vroomvroom).

This study uses the source repository as an example of remix-performance
architecture: named parts, stem roles, per-layer routing, and compact
performance states. It does not recreate the original remix.

## Boundary

Do not copy upstream audio, commercial remix material, or the source helper
environment into course work. The local `.tidal` file uses stock SuperDirt
sample names and generic musical roles so students can apply the architecture to
their own material.

## Teaching Role

Use this as an advanced extension after students are comfortable with layers,
`stack`, `struct`, routing controls, and manual arrangement states.

Start with
[`tidal/remix-performance-skeleton.tidal`](tidal/remix-performance-skeleton.tidal).
