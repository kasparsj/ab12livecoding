# Cross-Study Comparison Prompts

Use these prompts after students have tried at least one Kindohm study and one
ndr-brt study.

## Same Problem, Two Styles

| Prompt | Kindohm angle | ndr-brt angle |
| --- | --- | --- |
| Build a four-cycle transition. | Use `whenmod`, `trunc`, `stut`, and a return. | Use a section state and destabilize only one layer. |
| Make one sample feel like a whole track. | Scan and curate sample indices. | Build anchor, slow body, grain body, and accent body. |
| Create controlled instability. | Chain periodic transformations. | Restrict randomness to one dimension. |
| Build a melody from generated numbers. | Map `run` or choices into a scale. | Use a pseudo-Markov pattern or register layers. |
| Make a performance ending. | Drop layers with predictable cycle logic. | Rehearse section-state subtraction and global color change. |

## Reflection Questions

- What stayed fixed in each version?
- Which version was easier to recover from?
- Which version was more readable after five minutes?
- Which version sounded more performable?
- What would you remove before playing this live?

## Capstone Variants

- Five-day sketchbook: make five small files from one seed rhythm.
- Single-sample set: build a complete performance from one sample family.
- No-new-samples arrangement: use only `bd`, `sn`, `hh`, `cp`, and one melodic sample.
- One-pattern transformation performance: change form without replacing the base pattern.
